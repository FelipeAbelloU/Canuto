# =============================================================
# CANUTO — Extracción OCR de escaneados con docling
# =============================================================
# ENTORNO SEPARADO — NO instalar en el venv principal.
#
# docling exige transformers moderno (≥4.49) que ROMPE el stack de
# inferencia/entrenamiento del venv principal (transformers 4.46.3 + trl).
# Por eso vive en su propio entorno virtual, usado solo para convertir los
# PDFs escaneados/corruptos a Markdown (ver docs/comparacion_extraccion_md.md).
#
# Uso en la workstation (con GPU):
#   python -m venv venv-docling
#   venv-docling\Scripts\activate            (Windows)
#   source venv-docling/bin/activate         (Linux)
#   pip install -r requirements-docling.txt
#   # luego, desde el venv-docling:
#   python scripts/extract_text.py --ocr --ocr-engine docling
#
# Nota GPU: para acelerar en la RTX 4090, instalar torch con CUDA ANTES:
#   pip install torch --index-url https://download.pytorch.org/whl/cu121
# =============================================================
